JavaScript Helper Methods

Definition:

Helper methods are built-in JavaScript functions that help us interact with the user or the browser. They make tasks like displaying messages, getting input, or debugging easier.

Common Helper Methods:

  • document.write() - Writes content directly to the HTML document.
  • console.log() - Prints output to the browser console for debugging.
  • alert() - Displays a pop-up alert box with a message.
  • prompt() - Shows a pop-up input box and returns user input as a string.
  • confirm() - Shows a pop-up with OK and Cancel buttons, returning true or false.

Example:

Open the console (F12) to see outputs. Alerts and prompts appear in the browser.
