JavaScript Variables

Definition:

A variable is a container to store data values. In JavaScript, variables are used to store numbers, text, boolean, or other data types. Variables can be declared using let, const, or var.

Types of Variables:

  • let - variable that can change its value
  • const - constant variable, value cannot change
  • var - old way to declare variables (not recommended)
