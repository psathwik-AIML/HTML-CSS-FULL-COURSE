JavaScript Data Types

Definition:

Data types are categories of values in JavaScript that tell the computer what kind of data we are storing in a variable.

Types of Data:

  • Number - numeric values (e.g., 5, 3.14)
  • String - text enclosed in quotes (e.g., "Hello")
  • Boolean - true or false
  • Undefined - a variable declared but not assigned
  • Null - represents "no value"
  • Object - collection of key-value pairs
  • Array - ordered list of values

Example:

Open the console (F12) to see outputs of different data types.
