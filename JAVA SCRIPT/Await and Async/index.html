JavaScript Async/Await

Definition

Async/Await is a modern syntax in JavaScript to handle asynchronous operations. async makes a function return a Promise, and await pauses execution until the Promise resolves.

Key Points

  • Async functions always return a Promise.
  • Await can only be used inside async functions.
  • Makes asynchronous code look like synchronous code.

Example 1: Basic Async/Await


async function fetchData() {
    return "Data fetched successfully!";
}

fetchData().then(console.log);

Example 2: Async/Await with Promises


function getData() {
    return new Promise(resolve => {
        setTimeout(() => resolve("Task completed after 2 seconds"), 2000);
    });
}

async function asyncTask() {
    const result = await getData();
    console.log(result);
}

asyncTask();
