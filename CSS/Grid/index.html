CSS Grid

CSS Grid is a 2-dimensional layout system in CSS that allows you to arrange items in rows and columns. You can control the size, spacing, and alignment of items easily, and even overlap items. It is very useful for creating complex layouts like dashboards, galleries, and web pages.
