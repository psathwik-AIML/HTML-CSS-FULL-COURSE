Transform Property

Translate

translate() in CSS is used to move an element from its original position left, right, up, or down — without changing the layout of other elements.

Box 1

Scale

scale() in CSS is used to change the size of an element — make it bigger or smaller — without changing the layout of other elements.

Box 2

Rotate

rotate() in CSS is used to turn (spin or rotate) an element around its center point.

Box 3

Skew

skew() in CSS is used to tilt or slant an element along the X or Y axis
