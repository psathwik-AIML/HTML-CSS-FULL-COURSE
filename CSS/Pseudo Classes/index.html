Pseudo-classes


hover

The :hover pseudo-class is used to style an element when you move your mouse over it.

Ex:

mouse over me

active

The :active pseudo-class is used to style an element while it is being clicked (or activated).

Ex:

click and hold me

focus

The :focus pseudo-class is used to style an element when it receives focus, usually by clicking on it or tabbing to it.

Ex:

tap on me

first child

The :first-child pseudo-class is used to style an element that is the first child of its parent.

Ex:

  • apple
  • banana
  • mango
  • grapes

last child

The :last-child pseudo-class is used to style the element that is the last child of its parent.

Ex:

  • apple
  • banana
  • mango
  • grapes

nth child

The :nth-child(n) pseudo-class is used to style elements based on their position in the parent.

Ex:

  1. apple
  2. banana
  3. mango
  4. grapes
